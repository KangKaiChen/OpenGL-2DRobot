# OpenGL-2DRobot


2.(作業環境)
	作業環境:win11 64bit
	編譯器:Visual Studio 2022
  
3.(作業指定功能)
	已完成繪製2D機器人
	已完成設定按鍵並達成所要求的旋轉及伸縮功能(7個關節點,共25個按鍵)
4.(敘述如何操作我的作業)

   旋轉:
	頭部:設定兩個按鍵可達到頭部左轉及右轉(極限值為28度)
	手部:設定八個按鍵可達到左右手大小臂左轉及右轉(極限值為90度)
	腳部:設定八個按鍵可達到左右腳大小腿左轉及右轉(左右小腿向外旋轉給定了5度的極限值,其他則為90度)
   伸縮:
	手部:設定四個按鍵可達到左右手小臂伸長及縮短(小臂最短值為5,最長則無限制)
	腳部:設定兩個按鍵可達到左右腳小腿伸長及縮短(小腿最短值為5,最長則無限制)
   重置:
	設定一個按鍵可將機器人動作重置為原角度
